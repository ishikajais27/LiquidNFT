LiquidNFT Architecture & Theory

NFT-Collateralized Lending Protocol on Ethereum Sepolia — Detailed Reference

Overview LiquidNFT is an NFT-collateralized lending protocol on Ethereum Sepolia. It enables NFT holders to obtain ETH liquidity without selling their assets by using them as collateral.

Problem NFTs are illiquid — owners needing capital must sell, permanently losing ownership.

Solution A trustless pawn-shop model — lock NFT → borrow ETH → repay to reclaim. Defaulted loans trigger automated 24h English auctions.

Key Implementation Statements
Interest Buffer (0.1%)
Display = actual owed per-second accrual gap. Add 0.1% to amount to cover txn confirmation delay; retry on revert.
escrowId Generation
hashuser, nft, tokenId, timestamp. Same NFT redeposited yields unique IDs → sequential loans OK.
ETH as Strings
JS float + wei precision. BigInt via ethers.js; formatEther = string; persist as string only. NEVER cast to Number.
Batch Save Ops
/api/user/save multi-op per request. $set + $push + status updates in one call. Subdocs run first.
Lowercase Addresses
ETH addrs case-insens. Checksum = error data; hash & cipher store lowercased; compare both sides lowercased.
